Week 1: 02/03/09-06/03/09

02/03/09

Introduction to CARET and Sakai

When we arrived at CARET we got a short tour around the office by Katy and Stephanie. They showed us where we would work and which computers we will be using during our stay. When the tour was finished, we wrote a intruduction letter that we hang in the kitchen so everyone at CARET would be able to read it.

When we finisched our letter, Nicolaas gave us an introduction to Sakai, an open-source learning environment. It's a set of sofware tools that help instructors, researchers and students to collaborate online. It is runs Apache Tomcat that provides a Java HTTP web server environment for Java. On the front-end they mainly use jQuery, probably the best javascript library in the world that communicates through REST services with the back-end.

At the end, Nicolaas explained our first assignment. We had to develop a discussion widget that could be added on a page on Sakai. To make that widget, we had to use different kinds of technologies like Ajax and JCR, which is a type of Object Database tailored to the storage, searching, and retrieval of hierarchical data.

Afterwards we got started by installing sakai on our computers. You can download an adjusted version of the Quick Start guide.
